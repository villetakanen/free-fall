*An external processing unit, typically belt-worn or integrated into armor, running tactical analysis software.*

Connects via Neural Link (requires brainstem connection for full effect). Provides real-time combat telemetry, threat assessment, and predictive targeting solutions.

**Effect:** Grants 3 temporary Current Mind points per scene, usable *only* for combat-related checks (e.g., targeting, tactical awareness, initiative reactions). These points refresh at the start of each new scene where combat is anticipated.

**Binding rationale:** Mind 3 — constant processing of tactical data streams and managing the NAI assistant imposes a significant cognitive load.
