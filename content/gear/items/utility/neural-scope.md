*A weapon scope that interfaces directly with the user's nervous system via an arm-based Neural Link.*

Uses an NAI to analyze targeting data and provide micro-correction feedback directly to the user's muscles for enhanced accuracy.

**Effect:** Allows the user to reroll one die assigned to an attack action made with the attached weapon per turn.

**Binding rationale:** Mind 1 — processing the scope's targeting data and integrating the feedback requires mental focus.
