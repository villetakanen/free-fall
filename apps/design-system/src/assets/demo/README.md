# Demo Assets

This directory contains images used exclusively for **design system demos and development previews**. These assets are not shipped to production.

## AI-Generated Media

Some images in this directory are generated using AI tools (e.g., Google Gemini, Midjourney). These are clearly identified by their filenames, which include the generating tool's name:

- `kinetic-carbine-gemini-generated.png` — Generated by Google Gemini

### Policy

- AI-generated images are used **for demo and development purposes only** — never as final production art.
- All AI-generated files must include the tool name in the filename (e.g., `*-gemini-generated.*`, `*-midjourney-generated.*`).
- Production art must be original, commissioned, or properly licensed work.
- If an AI-generated image is later replaced with production art, the AI-generated file must be removed from the repository.
